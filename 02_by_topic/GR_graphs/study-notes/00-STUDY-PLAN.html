10-Day Study Plan (~2 hours/day)

Day 1: Graph Fundamentals — Representations & Traversals

📖 Read: 01-GR0-Graph-Fundamentals.md (Sections 1–4: Terminology through BFS) 📚 Reference: DPV Chapter 3.1–3.2

Focus:

  • Terminology: vertices, edges, directed vs undirected, degree
  • Adjacency list vs adjacency matrix — know the tradeoffs
  • BFS algorithm and its O(n+m) runtime
  • Dijkstra's as "BFS with weights"

✅ Checkpoint: Can you trace BFS on a 6-node graph and produce dist/prev arrays?

Day 2: DFS Deep Dive

📖 Read: 01-GR0-Graph-Fundamentals.md (Sections 5–8: DFS through Edge Types) 📚 Reference: DPV Chapter 3.2–3.3

Focus:

  • DFS for undirected graphs (connected components)
  • DFS for directed graphs (pre/post numbering)
  • The four edge types: tree, back, forward, cross
  • Key fact: G has a cycle ⟺ DFS tree has a back edge

✅ Checkpoint: Given a directed graph, can you run DFS and label every edge by type?

Day 3: Topological Sort + SCC Setup

📖 Read: 01-GR0-Graph-Fundamentals.md (Section 9: Topological Sort) + 02-GR1-Strongly-Connected-Components.md (Sections 1–3) 📚 Reference: DPV Chapter 3.3–3.4

Focus:

  • Topological sorting via DFS (sort by decreasing post number)
  • Only works on DAGs — why?
  • Definition of strongly connected components
  • The metagraph is always a DAG

✅ Checkpoint: Can you topologically sort a DAG by hand using post numbers?

Day 4: SCC Algorithm — The Main Event

📖 Read: 02-GR1-Strongly-Connected-Components.md (Sections 4–7) 📚 Reference: DPV Chapter 3.4

Focus:

  • The two-pass DFS algorithm for finding SCCs
  • Why we reverse the graph (swap sources/sinks)
  • The key lemma: highest post-number vertex is in a source SCC
  • Full worked example

✅ Checkpoint: Can you find all SCCs of a 10-node directed graph by hand? Can you explain why the algorithm works?

Day 5: 2-SAT

📖 Read: 03-GR2-2SAT.md 📚 Reference: DPV Chapter 3.4 (applications)

Focus:

  • CNF notation, k-SAT definition
  • Constructing the implication graph from 2-SAT clauses
  • The lemma: satisfiable ⟺ no x_i and ¬x_i in same SCC
  • The full algorithm: build graph → find SCCs → assign truth values

✅ Checkpoint: Given a 2-SAT formula with 4 variables, can you construct the implication graph, find SCCs, and determine satisfiability?

Day 6: MST — Kruskal's & The Cut Property

📖 Read: 04-GR3-Minimum-Spanning-Tree.md (Sections 1–5) 📚 Reference: DPV Chapter 5.1

Focus:

  • MST definition and tree properties
  • Kruskal's algorithm with Union-Find
  • The Cut Property (statement AND proof)
  • Why Kruskal's is correct (via Cut Property)

✅ Checkpoint: Can you state and prove the Cut Property? Can you trace Kruskal's on a weighted graph?

Day 7: MST — Prim's + Review

📖 Read: 04-GR3-Minimum-Spanning-Tree.md (Sections 6–7) 📚 Reference: DPV Chapter 5.1

Focus:

  • Prim's algorithm and its connection to Cut Property
  • Kruskal's vs Prim's comparison
  • Review all MST material

✅ Checkpoint: Can you explain how both Kruskal's and Prim's are special cases of the Cut Property?

Day 8: Markov Chains

📖 Read: 05-GR4-Markov-Chains-PageRank.md (Sections 1–5: through Ergodic)

Focus:

  • Markov chain definition, transition matrices
  • k-step transitions via matrix powers
  • Stationary distribution: πP = π
  • Ergodic = aperiodic + irreducible (strongly connected)

✅ Checkpoint: Given a 3-state Markov chain, can you write the transition matrix and compute P²?

Day 9: PageRank

📖 Read: 05-GR4-Markov-Chains-PageRank.md (Sections 6–9: PageRank onward)

Focus:

  • Citation count intuition → random surfer model
  • The damping parameter α and why it's needed
  • Handling sink nodes
  • The transition matrix for PageRank
  • Finding π in practice

✅ Checkpoint: Can you explain why the random surfer model guarantees ergodicity? What happens with sink nodes?

Day 10: Comprehensive Review

📖 Read: 06-Graphs-Cheat-Sheet.md — use this as your self-test

Activities:

✅ Final Checkpoint: Can you explain every algorithm's input/output/runtime without looking? Can you prove the Cut Property and the 2-SAT lemma?
